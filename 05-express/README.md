## APIRest

1. Clone the app
```
git clone https://github.com/jestrade/make-it-real-online.git
```

2. Get into the folder make-it-real-online/05-express
```
cd make-it-real-online/05-express
```

3. Install the app
```
npm install
```

4. Create a .env file
```
PORT=3000
```

5. Run the app
```
npm start
```
